### 準備階段（建議）

1. **檢查程式完整性**
   ```bash
   # 驗證所有文件存在
   ls -la game/ player/ ui/ utils/ main.py
   
   # 運行規格驗證
   python verify_spec.py
   ```

2. **準備三個 Demo 場景**

### Demo 場景 1: 隨機 AI vs 隨機 AI（快速演示）
```bash
python main.py --mode rr --no-initial-move --depth 1
```

**預期輸出**：
- 棋盤初始化
- 每回合顯示移動和時間
- 自動進行 20 回合
- 顯示最終勝者和耗時

**耗時**：約 10-15 秒

### Demo 場景 2: 人人對戰（互動演示）
```bash
python main.py --mode hh
```

**流程**：
1. 助教先輸入 UV 方初始異動（格式：`src_row,src_col,dst_row,dst_col`）
   - 例如：`4,4,5,5`
2. AB 方人類輸入棋步
   - 例如：`0,0,1,1` 表示從 (0,0) 移動到 (1,1)
3. UV 方人類輸入棋步
4. 循環進行

**時間承諾**：可在 120 秒內演示完整遊戲

### Demo 場景 3: AI vs AI（高級演示）
```bash
python main.py --mode aa --depth 2
```

**特點**：
- 展示高級 AI 對戰
- 自動完成整局遊戲
- 記錄詳細的計時和計分信息

**耗時**：約 20-30 秒（根據 AI 深度）

## 🚨 常見問題

**Q: 程式崩潰怎麼辦？**
A: 運行 `python verify_spec.py` 進行完整檢測

**Q: 輸入格式不對怎麼辦？**
A: 使用格式 `src_row,src_col,dst_row,dst_col`，例如 `0,0,1,1`

**Q: 想快速演示？**
A: 使用 `--mode rr --no-initial-move` 和 `--depth 1` 參數

**Q: 想演示智能 AI？**
A: 使用 `--mode aa --depth 3` 或更高深度
