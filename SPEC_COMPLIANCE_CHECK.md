✓ 完全符合

**時間檢查邏輯**:
- 超過 120 秒 → 立即判負 ✓
- 超過 60 秒 < 120 秒 → 消耗一次延長機會 + 計數超時次數 ✓

**計分檢查**:
- A, B, U, V = 3 分 ✓
- c, d, e, f, w, x, y, z = 1 分 ✓
- 吃掉棋子時加分 ✓

**勝負判定**:
```python
# 高獲勝優先級：獵殺分數 > 耗時
if self.ab_points > self.uv_points:
    return ("AB", "points")
elif self.uv_points > self.ab_points:
    return ("UV", "points")
else:
    # 同分時比較耗時
    if self.ab_time < self.uv_time:
        return ("AB", "time")  ✓
```
